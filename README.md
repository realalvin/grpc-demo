   * 项目Maven引入
     ```
       <!-- latest version-->
        <dependency>
           <groupId>com.google.protobuf</groupId>
           <artifactId>protobuf-java</artifactId>
            <version>4.0.0-rc-2</version>
        </dependency>
     ```
     * optional maven import:
     ```
       <dependency>
           <groupId>com.google.protobuf</groupId>
           <artifactId>protobuf-java-util</artifactId>
           <version>4.0.0-rc-2</version>
       </dependency>
     ```
